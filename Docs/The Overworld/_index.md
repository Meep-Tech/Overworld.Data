# Overworlds
An Overworld (In-Game), is the name given to the In-Game 2D [[Tile]]-based, character-playable companion to a [[../Chat/Overview|chat]] [[Room]]. The Overworld of a [[World Room]] is an customizeable area based on a [[World]] that players can navigate and interact with using controlable [[Character]] sprites.
